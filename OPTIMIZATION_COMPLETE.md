## 🏗️ 架构改进

### 模块化设计
```
npc_optimization/
├── context_compressor.py      # 上下文压缩
├── behavior_decision_tree.py  # 行为决策树
├── prompt_templates.py        # Prompt模板
├── memory_manager.py          # 记忆管理
├── react_tools.py             # ReAct工具系统
└── rag_memory.py              # RAG记忆检索
```

### 系统流程
```
决策流程:
1. 行为决策树（规则） → 日常行为（即时）
2. LLM决策（压缩上下文） → 复杂情况（1-3秒）
3. ReAct工具系统 → 自主调用工具
4. RAG记忆检索 → 获取相关记忆
5. 记忆管理 → 定期清理和压缩
```
